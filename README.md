# EvoWalker AI 🚶🧬

**EvoWalker AI** は、AI（自律型ロボット個体群）が「二足歩行」を自律的に学習し、世代交代を繰り返して「直進歩行技術」を磨き上げていく過程を、美しいビジュアルと数理的なデータで観察・分析できる**AI歩行進化シミュレーター（Webアプリケーション）**です。

就職活動における「Web開発実績」「プログラミング能力」「UI/UX・デザイン設計経験」をアピールするためのポートフォリオ成果物として開発されました。

---

## 🚀 プロジェクト概要

「機械学習の学習プロセス」を直感的に視覚化し、非技術者からエンジニアまで誰でも楽しめるように設計された、インストール不要のフロントエンド完結アプリです。
AIの歩行方針（安定重視、スピード重視、バランス）、学習スピード、同時歩行個体数、世代数をカスタマイズし、世代を追うごとにロボットたちがふらつきや転倒を克服し、まっすぐ安定して歩くようになる進化プロセスを観察・診断できます。

---

## 💡 解決したい課題 ＆ 就活でのアピールポイント

1. **AI学習プロセスの直感的可視化**:
   ディープラーニングや強化学習は結果だけが表示され、裏側の学習プロセス（なぜその挙動になるのか）がブラックボックス化しがちです。本アプリは、世代交代ごとの「スコアの向上」「安定歩行率の上昇」「転倒率の低下」をリアルタイムCanvasとSVGグラフで明示し、AIの「試行錯誤と進化」を誰もが体感できる教育デモに昇華させました。
2. **純粋な数理演算による2.5D投影 (描画アルゴリズム実装力)**:
   Three.js などの外部3D物理ライブラリに依存せず、**三角関数と遠近遠近パース投影（Pure Canvas 2D & Mathematics）のみで2.5Dの奥行き空間を構築**しました。エンジニアとしての数学的思考力と、スクラッチでの描画エンジン構築力をアピールしています。
3. **Figmaに準拠したクリーンなUI/UX設計 (デザインアピール)**:
   単に「動く技術デモ」に留まらず、余白、視線誘導、ボタン優先度を徹底設計した清潔感のある「ライトテーマ」を採用。シミュレーション中の「待機ストレス」を軽減する「早期終了条件」や「途中分析導線」を実装し、製品レベルのUX完成度を追求しています。

## 🌟 主な機能と実装工夫

### 1. 2.5D自律歩行シミュレーター (HTML5 Canvas 2D)
*   パース（奥に行くほど縮小）の効いた疑似3Dレーンに、立体的なゼブラフェンスやゴールゲート、進行方向を示す路面矢印を描画。
*   ロボットは頭部・胴体（LED付）・アーム・脚が動的に動くデザインを適用。`walker.id` に応じてアンテナやバイザーの形状にバリエーション（個性）を持たせています。
*   ロボットの進行速度やバランスの崩れに合わせ、**体の前傾姿勢（ピッチ）や左右への揺れ（ロール）が動的に傾く物理表現**を実装。
*   ゴール到達時には「王冠 👑」を掲げてバンザイして喜ぶアニメーションと、**「クラッカー紙吹雪（confetti粒子）」**の演出を行い、転倒時の「バタッと横倒しになり、星がくるくる回る気絶演出 💦」と対比させて成否を明確にしました。

### 2. 遺伝的アルゴリズム（Genetic Algorithm）
*   各個体の「最高速度制限」「旋回（重心）感度」「ブレーキ感度」「安全マージン」「歩行ブレ（ノイズ）」の5つの数理パラメータを「ゲノム（DNA）」として定義。
*   **適合度評価（Fitness Score）**: 前進した距離、生存フレーム時間、およびブレの小ささから「安定ボーナス」を加算し、転倒時には「ペナルティ」を引く厳密なスコアシステム。
*   **エリート選抜 (Elitism)**: 毎世代の上位20%をエリートとして選別し、その中でも最高成績を残したベスト個体のゲノムは無変更で次世代へ完全コピー（エリート保護）。
*   **突然変異 (Mutation)**: 残りの個体は選抜されたエリートを親とし、指定された学習スピードに応じた変異率（`5%〜30%`）で正規分布風のランダムノイズをゲノムに交ぜて次世代を生成。

### 3. 進化統計データのSVG可視化
*   「安定歩行率の推移」「平均前進距離の推移」「転倒率の推移」の3軸にデータを絞り込み、自作のSVGグラフで動的に描画。各グラフには指標の意味が直感的にわかる説明文を配備。
*   「第1世代 ➔ 最終世代」での数値の変化を一目で対比できる「進化比較テーブル」と、最終世代の成績上位3ロボットの詳細なスタッツ（ID、スコア、距離、生存時間、状態）を網羅したランキングリストをサイドに表示。

### 4. AI歩行診断レポート（ルールベース評価）
*   蓄積されたすべての世代結果を、ルールベース評価エンジンが解析。
*   適合プロセスから、最終歩行モデルを4つのタイプ（`進化成功型・総合自律歩行モデル`、`安定学習・リスク回避型モデル`、`攻撃的進化・ハイリスクハイピッチモデル`、`学習停滞・進化限界モデル`）に分類し、総合レーティング（S〜C）を算出。
*   「改善された点」「まだ残る弱点」「次回おすすめ設定」を結論ファーストで優先表示し、長文の詳細レポートはアコーディオン内に折りたたんで情報過多を防ぎました。

## 🎨 画面遷移 ＆ Figma設計意図

1.  **トップ画面 (`/`)**: アプリのイントロダクションとスタートCTA。学習プロセス（Gen 1 ➔ Gen 30）の変遷プレビューを添え、就活で見せる作品の「美しい表紙」としてヒーローセクションを最適化。
2.  **レーン選択画面 (`/scenario`)**: 直線レーン（広い/初級）、細道レーン（狭い/中級）、障害物レーン（立体ブロック障害物あり/上級）の3環境。
3.  **AI設定画面 (`/config`)**: AIの歩行方針（性格）、学習スピード、同時歩行個体数、最大学習世代数をカスタマイズ。
4.  **ブリーフィング画面 (`/briefing`)**: 選択された条件と、それによってAIの脳（初期パラメータ）にどんなバイアスが反映されたかを説明。
5.  **シミュレーション画面 (`/simulation`)**: 2.5D Canvasでの歩行シミュレーション。世代交代後のフリーズを防ぐ「React-Canvas状態同期ロック」を搭載し、高倍速（1x/2x/5x）切り替え、生存台数・安定歩行率のリアルタイムテレメトリ、3行スクロールロックログを完備。
6.  **世代分析画面 (`/analysis`)**: 各種統計指標のSVGグラフ、初期/最終世代の比較対照表。
7.  **最終診断画面 (`/result`)**: S〜Cの評価ランク、改善された点・弱点・おすすめ設定、および開発アーキテクチャ（GAの仕組み等）を語るアコーディオン詳細レポート。

## 🛠️ 技術構成 (Technology Stack)

*   **フロントエンド**: React 19.x, React Router v7.x
*   **ビルドツール / 言語**: Vite 8.x, TypeScript (Strict Mode)
*   **スタイリング**: Vanilla CSS (CSS変数を用いたクリーンなライトテーマ、Google Fonts Outfit & Inter)
*   **物理・描画ロジック**: Pure JavaScript & HTML5 Canvas API (数学的な2.5Dアフィン・パース変換)
*   **ビルド・型チェック**: `npx tsc --noEmit` & `npm run build` が100%エラーなく本番用にコンパイルされることを検証済み。

## ⚡ ローカル起動方法

事前に [Node.js](https://nodejs.org/) がインストールされていることを確認してください。

1.  **リポジトリのクローン**
    ```bash
    git clone https://github.com/your-username/evowalker-ai.git
    cd evowalker-ai
    ```
2.  **依存関係のインストール**
    ```bash
    npm install
    ```
3.  **開発サーバーの起動**
    ```bash
    npm run dev
    ```
    起動後、ブラウザで `http://localhost:5173/` を開きます。
4.  **本本ビルド**
    ```bash
    npm run build
    ```
    `dist/` ディレクトリに最適化された静的ファイルが出力されます。
