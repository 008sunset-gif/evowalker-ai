# Evodrive AI - Figma Plugin 利用手順書

本フォルダには、Figma上で『Evodrive AI』の主要7画面のレイアウト（Frame）を瞬時に自動生成するためのプラグインコードが格納されています。
ローカル環境でFigmaの成果物（ポートフォリオや発表資料用のデザインボード）を整備するための下地として活用してください。

> [!WARNING]
> 本プラグインやFigmaファイルは現在**ローカル準備段階**です。成果物全体の完成前にFigmaの「共有リンク（パブリックリンク）」を外部へ発行・公開しないように注意してください。

---

## 📂 フォルダ構成

*   `manifest.json`: Figmaプラグインの設定定義ファイル
*   `code.js`: Figma上で各画面のFrameとプレースホルダ要素を自動生成するJSスクリプト

---

## 🛠️ プラグインのセットアップ・実行手順

Figma（デスクトップアプリ版またはブラウザ版）を使用し、ローカルにあるプラグインフォルダを開発用プラグインとして読み込ませます。

### 1. Figmaで新規デザインファイルを作成
1.  Figmaを開き、右上の「**新規デザインファイル (New design file)**」をクリックして空のボードを作成します。

### 2. 開発者用プラグインとして読み込む
1.  左上のメニュー（Figmaアイコン）をクリックし、**「プラグイン (Plugins)」 ➔ 「開発 (Development)」 ➔ 「マニフェストのインポート (Import manifest from file...)」** を選択します。
2.  ファイル選択ダイアログが表示されるため、以下のパスにある `manifest.json` を選択します：
    `D:\projects\evodrive-ai\figma-plugin\manifest.json`
3.  インポートに成功すると、開発用プラグインの一覧に「**Evodrive AI Layout Generator**」が登録されます。

### 3. プラグインを実行する
1.  作成した新規ファイルのキャンバス上（何もない場所）で右クリックします。
2.  メニューから **「プラグイン (Plugins)」 ➔ 「開発 (Development)」 ➔ 「Evodrive AI Layout Generator」** をクリックします。
3.  実行すると、1440x900のデスクトップサイズで、以下の主要7画面のFrameが自動的に横並び（X軸ギャップ1600px）で生成されます。
    *   `01_Top` (起動画面)
    *   `02_ScenarioSelect` (コース選択画面)
    *   `03_AIConfig` (設定画面)
    *   `04_Briefing` (ブリーフィング画面)
    *   `05_Simulation` (シミュレーション走行画面)
    *   `06_Analysis` (推移グラフ分析画面)
    *   `07_Result` (総合運転診断画面)
4.  処理が完了すると、自動的にすべてのFrameが画面に収まるようにズームされ、プラグインが終了します。

## 🎨 生成後の手作業での微調整推奨ポイント

プラグインが自動生成するレイアウトは、Figma上の基本フォント（Inter）およびソリッドカラーのプレースホルダ（ワイヤーフレーム）です。Evodrive AIの「サイバーネオンSF調」を完成させるため、以下の手作業による調整を推奨します。

1.  **フォントの置換（デザイン仕様書準拠）**
    *   主要数値やテレメトリの値、タイトルなどのSF感を出したい箇所は、Figmaのフォント一覧から `'Orbitron'` （Google Fonts）へ手動で変更してください。
    *   その他のテキストは `'Outfit'` または `'Inter'` を設定します。
2.  **ネオングラデーションの適用**
    *   ボタンや特定の枠線（シアンやパープルの単色部分）に対し、Figmaの「Linear Gradient」を使用して `rgba(79, 209, 197, 1)` から `rgba(192, 132, 252, 1)` へのグラデーションを施すことで、実機UIに極めて近い光沢感を演出できます。
3.  **シミュレータープレビュー画像の追加**
    *   `02_ScenarioSelect` のコース画像プレースホルダや、`05_Simulation` のCanvas模擬枠の中に、実際のブラウザ画面からキャプチャした画像などを「塗り (Fill) ➔ 画像 (Image)」として流し込んでください。
4.  **グラスモルフィズム効果（背景のすりガラス）の付加**
    *   カードパネル（`Panel - ...`）のレイヤーに対し、レイヤー効果で **「背景のぼかし (Background Blur)」** を `12px` 〜 `20px` 程度適用すると、より高級感のある半透明UIに仕上がります。
