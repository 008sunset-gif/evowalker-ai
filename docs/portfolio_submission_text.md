## 面接で聞かれた時の説明文（口頭用・60〜90秒）

「EvoWalker AI は、遺伝的アルゴリズムで二足歩行ロボットが歩行を学習していく様子を観察できる、フロントエンド完結のWebアプリです。React と React Three Fiber でリアルタイム3D描画を行い、歩行の物理は外部の物理エンジンを使わず、サイン波ベースの歩容と重心バランスを自作で実装しました。

進化のロジックはニューラルネットワークや強化学習ではなく、遺伝的アルゴリズムです。各ロボットの歩き方を12個のゲノムパラメータで表現し、成績の良い上位個体を保護しつつ、突然変異と、毎世代少数のランダムな新規個体を加えて多様性を保ちながら改善させています。交叉はあえて使わず、挙動の説明性と実装の安定性を優先しました。

特に工夫したのは障害物コースで、『障害物をセンサーで検知して避ける』専用ロジックは作らず、蛇行や中央復帰といった歩行パターンの遺伝子の違いによって、結果的に避けられる個体が選抜されるようにした点です。失敗原因も、転倒・障害物衝突・コースアウト・立ち往生を別々に分類して可視化し、世代ごとに何が改善したかを説明できるようにしました。企画からUI/UX、3D、アルゴリズムまで一人で実装しています。」
